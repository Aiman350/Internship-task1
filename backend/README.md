

# Allied Software Engineers Internship Dashboard

A Full Stack MERN application developed as part of the Allied Software Engineers Internship Assessment.

## 🚀 Features

### Backend
- Express.js Server
- REST API Development
- Company Information API
- Intern Information API
- Internship Tasks API
- JSON Responses
- CORS Enabled

### Frontend
- React + Vite
- React Hooks (useState, useEffect)
- Fetch API
- Dynamic Data Rendering
- Responsive User Interface

## 🛠️ Tech Stack

- React
- Vite
- Node.js
- Express.js
- HTML5
- CSS3
- JavaScript (ES6)

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|----------|-------------|
| GET | / | Welcome Route |
| GET | /api/company | Company Information |
| GET | /api/intern | Intern Information |
| GET | /api/tasks | Internship Tasks |

## ⚙️ Installation

### Backend

```bash
cd backend
npm install
node server.js
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
